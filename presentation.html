Smart Version Detection

Understands Cargo's version requirement syntax

Exact vs Range Versions

Exact versions 1.2.3, 0.5.0

Compared directly against latest. If you specify 1.2.3 and latest is 1.2.4, you'll see üü® patch-behind.

Range versions 1, 1.2, ^1.2.3

Evaluated as ranges. If you specify 1 and latest is 1.9.0, you'll see ‚úÖ latest because 1.9.0 satisfies the range.
